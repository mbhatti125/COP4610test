Part 1

To compile the program type the make command in the terminal. 
To execute the program use "./myshell" into the terminal.
This will start the program and the shell. 
Enter your commands after the prompt. Use an ampersand (&) after a command to have it run in the background.
Use semicolon (;) to have several commands run concurrently. 
Use (|) between commands to pipe them. 

Examples of Commands: 

Exit the shell:
JohnSmith> quit

List files of the directory:
JohnSmith> /bin/ls
